PyBursa

В воскресенье, 5 марта 2017г., стала доступна первая неделя курса по веб-разработке с Python. Структурно курс PyBursa разбит на две части:

  • Python Basics
  • Django Basics

Вторая часть называется "Web разработка на Python/Django"

Технически курс организован в уже привычном для Вас формате. Теория в виде скринкастов и тестовые задания. Два раза в неделю будут открываться контрольные задания.

Материалы первой недели посвящены двум темам:

  • Основы html/css и bootstrap
  • Система контроля версий GIT

Кто незнаком ни с одной из тем будет сложно - материал достаточно обширный.

С другой стороны курс не ставит себе целью сделать из вас верстальщиков и не предполагает углубления в тему html/css, затронуты основы, которых достаточно для backend web-разработчика. Т.е. освещены все базовые принципы и подходы, но для расширения знаний и решения конкретных вопросов нужно будет пользоваться справочными материалами.

Чем лучше разработчик владеет таким инструментом как git тем, конечно, лучше. GIT - настолько могучая программа, что не существует пределов для совершенства в ее освоении! В материалах курса свещены основные принципы в объеме, достаточном для полноценной командной работы.

В любом случае жизнь разработчика связана с тем, что он сталкивается с темами которых он еще не знает (или забыл :) ) и нужно постоянно заниматься самообразованием, ну как минимум обращаться к документации за недостающим синтаксисом либо описанием работы функций, тех или иных модулей, команд и параметров. Советую привыкать к этому и смело ходить на ресурсы посвященные темам данной недели.
